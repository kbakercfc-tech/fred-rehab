Fred's Rehab App - How to Use

1. Enter Data

On the "Enter Data" page, you can log Fred's daily activities:

  • Date: Select the date for the entry.
  • Exercise Done: Choose 'Yes' or 'No'. If 'Yes', the 'Repetitions' field becomes active.
  • Repetitions: Enter the number of repetitions (defaults to 4 if 'Yes' is selected and not manually changed).
  • Steps Walked: Enter the total steps for the day.
  • Comments: Add any relevant notes or observations.

Use the "Save Exercise" and "Save Steps" buttons to save entries individually for the selected date. Be aware that saving new data for an existing date will overwrite previous entries for that type (exercise or steps).

2. Enter Data - Tabular

The "Enter Data - Tabular" page allows for quick entry of multiple days' data:

  • Select Date Range: Choose a start and end date (up to 30 days).
  • Click "Generate Table" to populate a table with the selected dates.
  • For each date, you can enter 'Exercise Done' (Yes/No), 'Repetitions' (defaults to 4 if 'Yes' and not changed), 'Steps' (simple text entry), and 'Comments'.
  • Click "Save All Data" to save all entries for the displayed dates simultaneously.

3. View History

The "View History" page displays a tabular list of all Fred's logged activities.

  • Share History: Use this button to share the history data via email or other applications (if supported by your browser).
  • Clear All History: This button deletes all exercise and step data from the database after a confirmation prompt. Use with caution!

4. View Steps Graph

Visualize Fred's steps progress over time:

  • Filter Options:
    • "All Entries": Shows data from the fixed start date (2025-10-24) to today.
    • "Last 30 Days": Displays data for the most recent 30 days.
    • "Select Period": Allows you to choose a custom start and end date for the graph. Click "Apply Filter" after selecting dates.
  • Chart Type: Toggle between "Line Chart" and "Bar Chart" views.
  • Legend: Understand the color coding for exercise status and largest steps day.
  • Summary Boxes: View "Largest Steps" and "Average Steps" for the displayed period.
  • Share Graph: Share a link to the graph page via email or other applications.

5. Calendar View

See Fred's activity at a glance in a calendar format. Days are color-coded based on activity levels (Green: Exercises & Steps Done, Amber: Exercises Done Only, Red: Nothing Entered).

6. Exercise Videos

Manage Fred's exercise videos:

  • Upload New Exercise Video: Provide a title and select a video file to upload.
  • Uploaded Videos: View all uploaded videos. Each video has a "Delete" button to remove it from the system (both database entry and the file).

Deployment & Data Persistence

This application is set up for deployment on Render. Your production database and uploaded videos are stored on a Persistent Disk on Render and are not overwritten by new deployments from your development environment, thanks to proper Git configuration (.gitignore) and environment variables.
